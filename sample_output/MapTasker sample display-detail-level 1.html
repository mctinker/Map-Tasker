  • Project: PrfTsk List
    • TaskerNet description:

    Profiles and Tasks Lister/LoggerList and optionally log all of your Profiles and Tasks.

    All functionality is self-contained in this project.

    Logging is done upon request by incorporating the functionality of TaskLogger and are logged to a shared public "Google Sheets" spreadsheet. If you wish to deploy your own spreadsheet, refer to TaskLogger here.

    This is an extension of this this Profile List, and has all of the functionality of this other project built-in, including copy-to-clipboard of selected profiles/tasks and invocation of selected profiles/tasks.

    After installing this project, add the task "PT List Profiles and Tasks" to your phone's desktop as a tasker shortcut and run it from there. You will be prompted for:

    - Once the profiles are listed, do you want to log them?

    - If you responded "yes" to logging, then you will be prompted for the sheet/tab name in the public spreadsheet to log to. Try to use a unique name to avoid stepping on top of someone else's log.

    - Once the tasks are listed, do you want to log them (via TaskLogger)?

    - If you responded "yes" to logging, then you will be prompted for the sheet/tab name in the public spreadsheet to log to. It can be the same name as used for your Profiles, or another name just for your Tasks.

    - If either is logged, do you want to wait for the logging to complete?

    This last prompt is provided since the logging could potentially take a minute or more. If you wait, it will show you the remaining log count every 5 seconds until the log is completed. If you don't wait, the task will end and the logging will complete in the background.

    Here's what you will see in this specific order:

    - A scrollable scene displaying all of your Profiles

    - Once you click on "Quit" or "X", you will get a prompt asking if want to log the results. If you respond "yes", you will be prompted for the spreadsheet sheet/tab name to use and the Profile-names logging will occur in the background.

    - A scrollable scene displaying all of your Tasks

    - Once you click on "Quit" or "X", you will get a prompt asking if want to log the results. If you respond "yes", you will be prompted for the spreadsheet sheet/tab name to use and the Task-names logging will occur in the background. you will be asked if you want to wait for the logging to complete. If you say "yes", then every 5 seconds you will be flashed a count of the remaining log entries to be processed. If you say "no", the task will exit and the completion of the logging will occur in the background.

    A second task is included, PT View Spreadsheet, which can be used to display the Google Sheet in a browser or via Google Sheets (if this app is installed on your phone).

    Note: Logging of say, up to 52 profiles and 140 tasks can take upwards of seven minutes or more. Check your output before rerunning this to make sure the previous run has completed.

    Note: You can view the spreadsheet here.
